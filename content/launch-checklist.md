# 🚀 SOCOS CRM — Launch Day Checklist

## Pre-Launch (24-48h Before)

- [ ] Final review of Reddit post copy
- [ ] Final review of DEV.to article
- [ ] GitHub repo is public and complete (README, license, contributing guide)
- [ ] Landing page is live with working waitlist form
- [ ] Discord server is set up with channels for feedback, support, contributions
- [ ] All social links are updated on GitHub profile

## Launch Day — Hour by Hour (PT)

### 6:00 AM — Final Checks
- [ ] Confirm all links work
- [ ] Prepare responses for "how is this different from X" questions
- [ ] Check GitHub repo one last time

### 7:00 AM — Reddit Post (r/startups)
- [ ] Post Reddit thread
- [ ] Set timer: monitor for 90 minutes
- [ ] Reply to first 10 comments personally
- [ ] Do not pin your own post

### 8:00 AM — HackerNews
- [ ] Post HN submission (Ask: "What's your system for staying in touch with people?")
- [ ] OR submit the GitHub repo if post is time-sensitive
- [ ] Check HN queue — engagement looks different, adjust tone

### 9:00 AM — Discord Announcement
- [ ] Post in #announcements
- [ ] Pin the launch post
- [ ] Welcome first new members personally

### 10:00 AM — Twitter/X Cross-Post
- [ ] Tweet: short hook + GitHub link
- [ ] Post in any relevant communities (indie hackers, bootstrapped, side project)
- [ ] Do NOT spam — meaningful engagement only

### 11:00 AM — DEV.to Article
- [ ] Publish DEV.to article
- [ ] Cross-post to HackerNews if platform allows
- [ ] Add to the article's Twitter thread

### 12:00 PM — Midday Check
- [ ] Update metrics tracking (GitHub stars, waitlist, Discord members)
- [ ] Reply to any outstanding comments on Reddit/HN
- [ ] Post a quick "thank you" comment to acknowledge early supporters

### 3:00 PM — Second Wave
- [ ] Post to r/SideProject
- [ ] Post to r/Entrepreneur (if you have post history there)
- [ ] Share in any relevant newsletters or communities

### 6:00 PM — Evening Wrap
- [ ] Check for any unanswered questions
- [ ] Update the Discord launch announcement with any news
- [ ] Save the Reddit post for future reference (screenshot, archive link)
