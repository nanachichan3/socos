## 🔧 PH Post Variation 2 — Technical/Open Source

**Angle:** For the HN/indie hacker/dev.to crowd who appreciates architecture and self-hosting

**Tagline:**
```
Open-source personal CRM with AI agents that remember your relationships — self-hosted, API-first, no vendor lock-in
```

**One-Liner:**
```
A personal CRM built for agents, not just humans.
```

**Description:**
```
Most CRMs are for sales teams. SOCOS is for humans.

Built with Next.js + NestJS + PostgreSQL. Fully self-hosted via Docker. REST + WebSocket API. AI agents run on Postgres + temporal.io — no proprietary AI stack, fully auditable.

Key features:
- Contact management with interaction logging
- AI agent that suggests who to reconnect with + why
- XP/levels/gamification for relationship maintenance habits
- Birthday reminders + lunar calendar support
- API-first architecture — build your own agent flows on top

40+ stars on GitHub. MIT license. Deploy in 5 minutes.

What would you want to build on top of a relationship API?
```

```
SOCOS is an AI-powered personal CRM that tracks every relationship you've ever made — with gamified XP, levels, and an agent that remembers why you met.
```
